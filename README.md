/ - 디렉터리의 이름 (명령프롬프트에서는 \ )

$ cd ~

$ pwd - 현재 디렉터리 경로 출력

$ mkdir test - 현재 디렉터리에 서브디렉터리 test를 만듬

$ cd test - 새로 만든 디렉터리로 이동

$ cd .. - 한 단계 위 디렉터리로 이동
